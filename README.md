## How to run

### 1. Clone Reporsitory

``` bash
git clone https://github.com/FabricioCes/task-developstoday.git
```
### 2. Go into project folder

```bash
cd task-developstoday
```

### 3. Install the required packages
``` bash
 npm i
```

### 4. Run the application

``` bash
 npm run dev
```
